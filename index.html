Cue Pin Select

The simple password manager with passwords that can be derived by hand.

About the App

The Cue Pin Select App is a simple password manager that generates passwords for services using a secret PIN, a secret passphrase, and a 4 letter cue specific to each service. Unlike with most password managers, the passwords can be quickly derived by hand if needed. The scheme will generate strong passwords that meet most common password requirements. Passwords are never stored, but instead are derived when needed from the stored secret PIN and passphrase. The app supports Face ID for quick access after initial setup.
